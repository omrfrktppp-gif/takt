# Görseller — Alt Text Manifesti

Bu yazıya ait tüm görseller bu klasördedir. Her görsel için dosya adı + içerik + alt text aşağıda listelenir (görsel SEO, rehber §9.10).

| Dosya | İçerik | Alt text |
|---|---|---|
| `cover.jpg` | Kapak görseli | |

## Kurallar
- Dosya adı: küçük harf, tireli, anahtar kelimeli. Türkçe karakter yok.
- Format: fotoğraf için `.webp`/`.jpg`, diyagram için `.svg`/`.png`.
- Kapak: `cover.jpg` (veya `.webp`), geniş en-boy (16:9 önerilir).
- Görseller deploy öncesi sıkıştırılır (WebP, rehber §10.5).
